# Code Of Conduct

This project follows a simple contributor standard:

- Be respectful.
- Assume good intent, then clarify.
- Critique content, not people.
- Keep feedback specific and useful.
- Do not harass, threaten, insult, or discriminate.
- Do not submit exam dumps or confidential certification material.

Maintainers may remove comments, issues, pull requests, or contributors that make the project hostile, unsafe, or legally risky.

If you need to report a conduct issue, open a private security advisory if available or contact the repository maintainer through GitHub.
