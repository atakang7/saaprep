# Security Policy

This repository contains study documentation and practice material. It should not contain production secrets, credentials, or private infrastructure details.

## Reporting A Problem

Please report security or policy-sensitive issues privately when possible, especially:

- Accidentally committed credentials or private account details
- Copyrighted practice questions or exam dumps
- Confidential exam content
- Malicious links or unsafe commands

Use GitHub private vulnerability reporting if enabled, or contact the maintainer through GitHub.

## Supported Content

The current maintained content targets AWS Certified Solutions Architect - Associate SAA-C03. AWS may update exam scope or service names, so corrections are welcome.
