# Sample Exams

This folder is for legitimate free SAA-C03 sample material.

Downloaded files:

- [aws-official-saa-c03-sample-questions.pdf](aws-official-saa-c03-sample-questions.pdf): official AWS sample exam questions PDF. Use this as a short timed warmup. The answer key is at the end of the PDF.
- [aws-official-saa-c03-sample-questions-answer-key.md](aws-official-saa-c03-sample-questions-answer-key.md): extracted answer letters with short study notes for checking yourself after the PDF.
- [aws-official-saa-c03-exam-guide.pdf](aws-official-saa-c03-exam-guide.pdf): official AWS exam guide PDF. This is not a practice exam, but keep it beside the samples so you can check scope.

Online official practice:

- AWS Skill Builder official practice question set: https://explore.skillbuilder.aws/learn/courses/13266/exam-prep-official-practice-question-set-aws-certified-solutions-architect-associate-saa-c03-english

Use the Skill Builder set when you want another official 20-question check. It may require a free AWS Skill Builder login, so it is linked here instead of copied into the repo.
